Java Servlets

¿Qué son los Java Servlets?

Los Java Servlets son una tecnología de Java que permite extender la funcionalidad de servidores web para generar contenido dinámico.

Los servlets son programas Java que se ejecutan en un servidor web y generan respuestas HTML que pueden incluir datos obtenidos de bases de datos, archivos o cualquier otro recurso.

Además, los servlets tienen la capacidad de manejar eventos del lado del servidor, como solicitudes de formularios y solicitudes de información, y enviar la respuesta correspondiente al cliente.

Práctica de Servlets

En la práctica de servlets se implementó un CRUD para la página de registro de ESCOM Groups. El objetivo era permitir a los usuarios crear una cuenta en el juego, ver el ranking de los jugadores y actualizar su información personal.

Para ello se crearon varias clases servlets que manejan las solicitudes HTTP del usuario, las cuales se comunican con una base de datos MySQL para realizar las operaciones de creación, lectura, actualización y eliminación de datos.

El resultado es una página web dinámica que permite a los usuarios registrarse en el juego y actualizar su información personal, además de ver el ranking de los jugadores.

Creación y eliminación de cuenta

Para crear una cuenta en ESCOM Groups, el usuario debe llenar un formulario con su nombre de usuario, correo electrónico y contraseña. Una vez que se envía el formulario, se crea una nueva entrada en la base de datos con los datos del usuario.

Además, se implementó una opción para eliminar la cuenta del usuario. Para ello, se creó un servlet que elimina la entrada correspondiente en la base de datos al recibir una solicitud HTTP DELETE.

Ranking de jugadores

La página de registro de ESCOM Groups también incluye una sección para ver el ranking de los jugadores. Para ello, se creó un servlet que obtiene la información de los jugadores de la base de datos y los ordena según su puntaje.

La información se presenta en una tabla HTML dinámica que se actualiza automáticamente cada vez que se carga la página o se realiza una operación de creación, actualización o eliminación de datos.

Actualización de información personal

Para permitir a los usuarios actualizar su información personal, se creó un formulario que se llena con los datos actuales del usuario. Al enviar el formulario, se realiza una operación de actualización en la base de datos para cambiar los datos del usuario.

Además, se implementó una opción para cambiar la contraseña del usuario. Para ello, se creó un formulario que solicita la contraseña actual y la nueva contraseña. Al enviar el formulario, se realiza una operación de actualización en la base de datos para cambiar la contraseña del usuario.
